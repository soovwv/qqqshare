# Code of Conduct

## Our standards

QQQShare contributors and maintainers commit to a respectful, welcoming, and harassment-free community. Be constructive, assume good intent while addressing impact, respect privacy, and focus criticism on ideas and code rather than people.

Unacceptable behavior includes harassment, discrimination, threats, sexualized attention, sustained disruption, publishing another person's private information, or exposing share URLs, tokens, or files without permission.

## Enforcement

Maintainers may edit or remove contributions, comments, issues, and other content that violates these standards, and may temporarily or permanently restrict participation. Enforcement decisions should be proportionate, private where possible, and documented for maintainers.

Report conduct concerns privately to the repository owner using the contact method listed on the owner's GitHub profile. Do not open a public issue containing private details. GitHub platform abuse may also be reported directly to GitHub Support.

This project policy applies in QQQShare project spaces and when a person is officially representing the project.
