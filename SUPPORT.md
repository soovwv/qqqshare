# Support

QQQShare is community-maintained open-source software and is provided without warranty or guaranteed response times.

- Bugs and reproducible feature requests: GitHub Issues
- Usage questions and ideas: GitHub Discussions, when enabled
- Security vulnerabilities: private reporting described in [SECURITY.md](SECURITY.md)

Before requesting help, use the latest release, confirm both devices are on the same trusted private network, check the Windows private-network firewall permission, and include the operating system and QQQShare version. Remove file names, IP addresses, tokens, and other sensitive data from screenshots and logs.
